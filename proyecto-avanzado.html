Proyecto Avanzado

Aplicación web full-stack con funcionalidades en tiempo real, roles de usuario y despliegue continuo en AWS.

Live Demo Código en GitHub

Capturas de pantalla

Características Destacadas

  • Actualizaciones en tiempo real con WebSockets.
  • Autenticación y autorización con JWT y middleware de roles.
  • Interfaz responsiva construida con Tailwind CSS.
  • Backend en Node.js + Express, base de datos MongoDB.
  • Despliegue continuo en AWS usando GitHub Actions y Docker.

Arquitectura del Proyecto

Frontend (React) ↔ API Gateway (Express) ↔ MongoDB Atlas
         │
     WebSockets (Socket.io)
         │
          → Auth Service (JWT)
         │
     CI/CD (GitHub Actions → AWS ECS/Docker)

Tecnologías Utilizadas

React Node.js MongoDB Tailwind CSS Socket.io GitHub Actions Docker
